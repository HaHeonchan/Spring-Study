# Spring Study

> 그동안 야매로 사용했던 스프링의 기초를 체계적으로 학습하기 위한 레포지토리.

---
## 학습 목표

* 스프링의 핵심 원리(IoC, DI, AOP)를 명확히 이해하기.
* 스프링 컨테이너가 빈을 어떻게 관리하는지 파악하기.
* 테스트 코드 작성법 익히기.
